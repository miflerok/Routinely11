Routinely - Трекер привычек

О проекте:

Routinely - это минималистичное и удобное мобильное приложение для формирования и отслеживания ежедневных привычек. Создано в рамках курса "Разработка мобильных приложений".

Основные возможности:

1. Простота использования: Добавление и отметка привычки всего в несколько кликов.
2. Визуализация прогресса: Наглядные графики и статистика для отслеживания достижений.
3. Полная оффлайн-работа: Приложение не требует постоянного подключения к интернету.
4. Минималистичный дизайн: Интуитивно понятный и приятный интерфейс.
5. Управление привычками: Создание, редактирование и просмотр списка привычек.
6. Отметка выполнения: Возможность отмечать выполнение как тапом, так и с помощью ползунка для количественных привычек.
7. Статистика и аналитика: Просмотр общего процента выполнения, календаря с отметками и лучших серий (стриков).
8. Настройки: Управление уведомлениями, выбор темы (светлая/темная/системная) и другие персонализированные опции.

Технологический стек:

•	Язык программирования: Kotlin

•	Архитектура: MVVM (Model-View-ViewModel)

•	База данных: Room (локальное хранение данных)

•	Дизайн: Material Design 3

•	Платформа: Android 9+

## Как запустить тесты

1. Убедитесь, что установлены JDK 17+ и Android SDK (platforms и build-tools, указанные в `app/build.gradle.kts`).
2. Выполните команду для сборки и прогона всех unit-тестов:
   ```bash
   ./gradlew test
   ```
3. Если сборка впервые и Gradle не скачивает зависимости из-за прокси/сети, попробуйте повторно запустить команду после настройки доступа или используйте заранее прогретый Gradle-кэш.
4. Для чистого прогона после смены зависимостей можно выполнить:
   ```bash
   ./gradlew clean test
   ```
